This is an interactive tutorial, which means you will be able to use R without installing anything! 

To use R, you will find a series of code exercises. In these exercises you will be asked to fill missing element by replacing the _ _ _ or to write the code from scratch in the INPUT boxes. Once you think you have the right answer, just click on **"Run Code"** to see the results. 

Note: The text starting with `#` are not run by R, thus they are used to comment the code and to give advices.
